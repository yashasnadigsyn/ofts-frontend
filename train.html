OFTS-Cli User Guide

A simple way to manage your photo collection from your terminal.

Introduction

OFTS-Cli is a command-line tool that helps you manage your photo collection right from your terminal. It uses DeepFace for face recognition and Microsoft's GIT for image captioning.

Installation

  1. Clone this repository: 
    git clone https://github.com/yashasnadigsyn/ofts-cli
cd ofts-cli
  2. Create a virtual environment: 
    python3 -m venv OFTS_CLI_VENV
source OFTS_CLI_VENV/bin/activate
  3. Install the required packages: 
    pip install -r requirements.txt
  4. Run the CLI: 
    python3 ofts_cli.py

Usage

After running OFTS-Cli, it will process your photos and generate a JSON file by recognizing faces and captioning them. You need to place this JSON file in the OFTS frontend folder.
